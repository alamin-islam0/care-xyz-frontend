# Care.xyz - Trusted Caregiving Services Platform

![Care.xyz Banner](https://via.placeholder.com/1200x600/0d9488/ffffff?text=Care.xyz+Platform)

Care.xyz is a modern web application designed to connect families with verified, professional caregivers. Whether you need baby sitting, elderly care, or special needs support, our platform makes finding and booking trusted care simple and secure.

## üì¶ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/alamin-islam0/care-xyz-frontend.git
   cd care-xyz-frontend
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root directory and add:

   ```env
   NEXT_PUBLIC_PUBLIC_API_URL=http://localhost:5000/api/v1
   ```

   _(Or point to your deployed backend URL)_

4. **Run the development server**

   ```bash
   npm run dev
   ```

5. **Open locally**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
